# Birbnb - Plataforma de Reservas de Alojamientos

Birbnb es una aplicación web full-stack que permite la gestión de reservas temporales. Este proyecto fue desarrollado como parte de la cursada de Desarrollo de Software.

## 📺 Demo

[![Ver Demo](https://img.shields.io/badge/Demo-Reproducir_Video-red?style=for-the-badge&logo=youtube)](https://www.youtube.com/watch?v=G992bgjGc2s)

---

## 🛠️ Tecnologías utilizadas

- **Frontend:** React.js, JavaScript, CSS3.
- **Backend:** Node.js, Express.
- **Base de Datos:** MongoDB (Mongoose para el modelado).
- **Arquitectura:** Microservicios / MVC (según corresponda tu caso).

## 🚀 Guía de Instalación Local

Seguí estos pasos para levantar el proyecto en tu computadora.

### 1. Requisitos previos
- Tener instalado [Node.js](https://nodejs.org/).
- Tener una instancia de [MongoDB](https://www.mongodb.com/try/download/community) corriendo localmente (puerto 27017) o una URI de MongoDB Atlas.

### 2. Clonar el repositorio
```bash
git clone [https://github.com/TU_USUARIO/birbnb.git](https://github.com/TU_USUARIO/birbnb.git)
